# GhostLogin
GhostLogin is an educational cybersecurity project that automates SSH exposure testing.

## Features 
- Detects hosts with open SSH using Nmap
- Tests credentials using Hydra
- Automatically verifies access with SSH 
- Generates logs and reports

## Technologies 
- Bash
- Nmap
- Hydra
- SSH / sshpass
  
## Usage
Run the script:
```bash
chmod +x ghostlogin.sh
./ghostlogin.sh
